So far the only way I found to extract in a single step is to create a tar using 7za and using the commands below. The extraction below doesn‚Äôt create any more temporary files (whonix.tar) and takes about 2 minutes for me currently. However srep by default requires 928 MB RAM for decompression of whonix.tar.srep. There is another option (srep -m3o whonix.tar) which will require almost no memory during decompression but it extends decompression time, for me it took a little longer than 3 minutes.

compression:
7za a -ttar whonix.tar gateway.ova workstation.ova
srep whonix.tar

decompression:
srep -d whonix.tar.srep - | 7za x -ttar -si

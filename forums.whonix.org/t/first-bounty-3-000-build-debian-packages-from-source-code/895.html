Gentoo has been investigated:

GitHub

Whonix/Gentoo-Port

Contribute to Whonix/Gentoo-Port development by creating an account on GitHub.

Gentoo would solve one issue, but generate new issues.

  • It’s a rolling distribution. Changing too rapidly. While Whonix was based on testing for a while, many issues were generated from that. Stable distributions are much more usable.
  • There are package manager and other security issues with Gentoo’s, see: https://github.com/Whonix/Gentoo-Port/labels/security
  • porting to Gentoo would cost lots of time and effort

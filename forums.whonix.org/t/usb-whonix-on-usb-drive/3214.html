Interestingly, for this type of installation, the Qubes platform is much simpler to set up than Virtualbox since the host configuration is included and all the Whonix VMs are packaged together. I have not tested this but you would probably want to use a quick USB3 drive of at least 32 GB. You should be fine following the standard setup instructions: https://www.whonix.org/wiki/Qubes after installation of Qubes to USB. Please report on your experience if you can.

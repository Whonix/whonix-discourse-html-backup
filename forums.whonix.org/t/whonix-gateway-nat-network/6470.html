When you attach the USB to the VM there should be nothing to change on the host network. I can‚Äôt test it atm, but when you attach the USB you should get a new network interface. So first remove the first network interface of the VM, then connect the USB adapter and boot it. You should have two interfaces in the VM then with eth0 hopefully being the USB adapter. Then change the static address for eth0 in /etc/network/interfaces.d/30_non-qubes-whonix as well as the gateway.

Whonix has been accepted into Google Season of Docs 2019!

News
0brand May 4, 2019, 6:27pm #1

We have more great news to share with our community. The Whonix project has been selected to participate in Google Season of Docs (GSoD) 2019! As a recap, GSoD is Google’s annual program that fosters collaboration between open-source project and tech writers. The goal of this program is to provide a framework for technical writers and open source projects to work together towards the common goal of improving an open source project’s documentation.During the program, technical writers spend a few months working closely with an open source community. They bring their technical writing expertise to the project’s documentation, and at the same time learn about the open source project and new technologies.

Interested technical writers can find exciting project ideas on our dedicated Season of Docs wiki page. The technical writer’s guide contains all the information you need to get started on your way to become a technical writer for the Whonix project.

If you have any questions please don’t hesitate to ask. And remember, we are looking for both skilled and enthusiastic technical writers. Anything you can do to set yourself apart from the other applicants would be in your favor. Most of all, be proactive in your approach, we want to see that you’re excited to collaborate with Whonix.

If you have the same error, that‚Äôs your kernel has the options mentioned above disabled.
Checkout documentation for your Linux Distribution on how to configure and recompile kernel.
You must have

CONFIG_CFQ_GROUP_IOSCHED=y
CONFIG_BLK_DEV_THROTTLING=y
In the ‚Äúmake menuconfig‚Äù you can press / button to find where these options are.

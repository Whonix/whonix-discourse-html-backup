Tor Browser Downloader (Whonix) download error for version 6.5a1

Support
gumshoe June 16, 2016, 9:20pm #1

Greetings and salutations Whonix Forum folks,

I’m experiencing something of a hiccup when using the Tor Browser Downloader by Whonix (in the Whonix-Workstation TemplateVM and Whonix-Workstation appVMs in Qubes-Whonix).

There are presently three ‘Online versions’ (6.0.1, 6.5a1, and 6.5a1-hardened) of the Tor Browser helpfully listed for download from the Tor Browser Downloader.

If I select and attempt to download the 6.5a1-hardened version, I get a pop-up error window saying, “ERROR: Failed to download: https://dist.torproject.org/torbrowser/6.5a1-hardened/sha256sums.txt.asc”.

If I select and attempt to download the 6.5a1 version, I get a pop-up error window saying, “ERROR: Failed to download: https://dist.torproject.org/torbrowser/6.5a1/sha256sums.txt.asc”.

If I select and attempt to download the 6.0.1 version, there’s no error message, the download proceeds splendidly, and the installation completes successfully.

I could be off base, but I suspect the above two error messages may be related to the intended “sha256sums.txt.asc” file actually being titled “sha256sums-unsigned-build.txt.asc” on The Tor Project’s website.

The correct file locations appear to be:

6.5a1-hardened version:
https://dist.torproject.org/torbrowser/6.5a1-hardened/sha256sums-unsigned-build.txt.asc

6.5a1 version:
https://dist.torproject.org/torbrowser/6.5a1/sha256sums-unsigned-build.txt.asc

Anyway, I just wanted to give the developers a heads-up on the off chance they weren’t already aware.

I’m not sure if I’m the only one experiencing this issue, or if there’s any way for me to fix it (other than manually downloading [bypassing the ‘Tor Browser Downloader by Whonix’] the 6.5a1 or 6.5a1-hardened versions, or simply resigning myself to using the 6.0.1 version)?

Thanks for your time and attention. And thanks for the nifty operating system.

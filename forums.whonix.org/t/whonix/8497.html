whonix设置系统中文和安装中文输入法教程

汉语/漢語 / Chinese
whbbk955 November 17, 2019, 4:08pm #1

因为whonix从15.X版本开始，更换了桌面系统。
官方的中文教程已经失效。
为了方便新手设置whonix新的15.X版本，
让系统可以显示中文，并安装中文输入法。符合正常中国人需求。
做个教程，方便大家。
（首先声明，本人也是新手，以下教程是琢磨了几天研究出来的，如果有什么不对的，欢迎大家讨论，更欢迎高手做更好的教程）

先安装whonix，并按照正常教程开启2个虚拟机。
（国内洋葱网络无法连接，建议在真实机器上开启SSR客户端，并在SSR设置中开启允许局域网访问，然后在网关机-gateway中设置TOR前
置代理，IP填写真实机器的IP，端口是1080即可）

然后进入正常使用的虚拟机-workstation，点击左下角，第4个黑乎乎的终端图标。
打开终端，
输入命令
sudo su

然后输入默认密码
changeme

再输入命令（依次一行行的输入进去）
apt-get update
apt-get install locales
dpkg-reconfigure locales

跳出一个界面，按方向键下，一直按到最下面，再往上翻几个，
用空格选择zh_CN UTF-8 UTF-8（选择好了，会有一个*号）
然后回车，又弹出一个窗口，用方向键选择zh_CN UTF-8。然后回车

输入命令（依次一行行的输入进去，有询问的就输入Y，然后回车）
apt-get install ttf-arphic-uming
apt-get install ttf-wqy-zenhei
sudo apt install fcitx
sudo apt install fcitx-googlepinyin
reboot

以上命令输入完毕，系统会自动重启，请等待进入系统。
进入系统以后，
在右下角那个键盘图标上点击右键，
选择—配置—点击+号----选择Google拼音—确定–关闭这个窗口

至此，系统中文和中文输入法已经设置完毕。

因为TOR浏览器默认会关闭js，导致某些网站无法注册和登陆（包括WHONIX官网）
所以说一下如何设置TOR浏览器为中文，和如何开启，关闭JS。
不过对于隐私设置方面，是否会有影响，还请高手指点
（有需要的按照以下教程设置即可）
打开tor浏览器—浏览器右上角3个横线的图标—选择Preferences—往下拉—点击English(United States)—点击Search for more
languages—稍等一会，点击 Select a language to add …—选择chinese(china)—点击右边的Add—稍等一会，等待下载—确认
光标已经选择了chinese(china)—点击 OK—点击（应用并重新打开）

这时候，tor浏览器已经变成中文了，如果需要打开JS，请跟着下面操作

点击游览器右上角3个横杠的图标—点击（首选项）—点击左边的（隐私与安全）—往下拉，在安全等级这个部分，根据你的需要选择
标准，中等，或者最高安全性。

至此，基本都是中文了，基本应用是没问题了。

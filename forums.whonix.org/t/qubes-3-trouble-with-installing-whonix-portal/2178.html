toechief:

use lspci in dom0 Konsole to figure out what that device is. Probably network adapter but could be graphics, usb, etc.

Do you have network connectivity at all?

Qubes App Menu -> sys-net -> launch Terminal
ping -c 3 fedoraproject.org

  1. If no network, search terms “fedora [your network adapter]”
  2. still need help, ask http://fedoraforum.org/
  3. still need help, ask https://groups.google.com/forum/#!forum/qubes-users

If your network is working, update your TemplateVMs and dom0 and try to get Whonix Templates again.

with this.

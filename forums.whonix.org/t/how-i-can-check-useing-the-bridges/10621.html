johny:

How it work? Bridges have auto changes? Or i must set this every time, every launch whonix?

Bridges stay set until you change the tor settings

johny:

And how i can check useing the bridges in real time? I want make sure that this work right.

look into the onion circuits gui and if the name of the first node isn’t obvious as being a bridge, check its IP on https://metrics.torproject.org/rs.html and it’ll say what node type it is.

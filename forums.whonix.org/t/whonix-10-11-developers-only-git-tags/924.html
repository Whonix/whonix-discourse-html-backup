[font=arial]–tb closed[/font] is another optional simplification. It installs Tor Browser by default in Whonix-Workstation. “closed” means it would fail closed and show and error during build if installation falls. “open” would ignore this and continue. You’d have to check the right version was installed during build. (By checking the build log or by checking the [font=arial]/home/user/.tb/[/font] folder. It’s just an option for custom builds, developers and derivatives. Not for official Whonix redistribute builds (reasons: https://www.whonix.org/wiki/Tor_Browser#Not_installed_by_Default).

If you’re wondering what the “-E” in “sudo -E” is about… I am running “export http_proxy=http://lan-ip:3142” beforehand where apt-cacher-ng is running on a different system to speed up builds.

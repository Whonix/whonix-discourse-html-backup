Major Changes

  • OpenSSL security fix for Debian and Tor Browser (OpenSSL 1.1.1d) (CVE-2021-3449)
  • Tor Browser 10.0.15
  • monero-gui v0.17.1.9
  • no longer install qtox by default Thanks to @madaidan for the suggestion! https://forums.whonix.org/t/tox-qtox-whonix-integration/1219/18
  • New anon-log command line utility.
    • anon-log simplifies the manual dump Tor log command by only showing relevant log entries and no non-issues.
  • New anon-consensus-delete command line utility.
    • Tor Consensus Re-Download
    • While this persists Tor Entry Guards, it results in Tor re-downloading the Tor Consensus document. Should usually not be required.
  • disable the following Tor stream isolation ports
    • privoxy 9112
    • polipo 9113
    • Tor Browser Updater by Whonix Developers gpg key download 9116 (key hardcoded in package source code)
    • TorChat 9119
    • mixmaster update 9120
    • mixmaster 9121
    • Tor Messenger 9153
    • since deprecated applications in Whonix and no longer in use
  • /usr/lib/anon-gw-anonymizer-config/torrc-d-cleaner: fix Avoid moving /etc/torrc.d/95_whonix.conf to /etc/torrc.d/backup/95_whonix.conf.dpkg-new during anon-gw-anonymizer-config package upgrades.
  • install no longer no longer required auditd by default https://phabricator.whonix.org/T537
  • hardened-malloc-type-test
  • add hardened-malloc-enabled-test
  • add new tool apparmor-watch
  • add minimum-time-check
  • constrained system resources program starter wrapper
  • Create apparmor-info (Thanks to madaidan!)
  • add /usr/bin/apt-get-reset

build system:

  • all packages are more standalone, no Whonix specific build dependencies (genmkfile) (packages can now be build without genmkfile)

Whonix VirtualBox:

  • Recommended VirtualBox version: 6.1.18
    • At time of writing.
    • Above link takes precedence.

onion-grater:

  • Improve systemd sandboxing (Thanks to madaidan!)
  • simplify 40_bitcoind.yml (Thanks to qubenix!)
  • update 40_bitcoind.yml to work with bitcoind v0.21 (Thanks to qubenix!)

Qubes-Whonix:

  • install vm-config-dist also in Qubes-Whonix to have QMLSCENE_DEVICE=softwarecontext workaround available

sdwdate:

  • improved sandboxing (Thanks to madaidan!)
  • refactoring
  • python pep8
  • update onion time sources (Thanks to TNT!)
  • increase timeout to 120 seconds due to slow onions
  • sdwdate Time Replay Protection
  • document sdwdate Clock Randomization
  • document sdwdate Time Sources Criteria (Thanks to TNT!)

whonix-welcome-page:

  • add link to https://web.archive.org/https://t.me/s/Whonix_All_News

  • split Anon Meta Packages into Anon Meta Packages (Whonix) and Kicksecure Meta Packages

Not immediately helpful to point out but a blinking underscore isn’t the same as a black screen. A black screen is an issue with drawing at all. A graphic card driver / kernel / virtualizer issue.

A blinking underscore is evidence of a functional graphical output.

That might help should you decide to research this issue further.

An easy fix - data loss inside VM - if appropriate (not caring about data loss inside VM):

Kicksecure – 21 Dec 21

Kicksecure ™ Factory Reset

Resetting Kicksecure ™ to Vendor Defaults. Reset configuration files to defaults, re-install meta package, move the system closer to vendor defaults.

(Whonix is based on Kicksecure.)

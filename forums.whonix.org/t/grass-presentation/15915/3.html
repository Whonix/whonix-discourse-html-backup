Just copy the PGP MESSAGE BLOCK and the PGP PUBLIC KEY BLOCK, import the public key and verify and run gpg -d file.

GPG names are confusing but that is a signature made with --sign and --armor. If it was made with --clear-sign, it would show what you normally expect PGP SIGNED MESSAGE. Because of pgp formatting, I couldn’t include the public key and make a clear sign because pgp itself changes the header - --- for PGP delimiters, making the message be verified but the public key not being able to import.
If on the other hand, I didn’t sign the public key, which was not necessary because the message is signed by it, then --clear-sign would be possible.

Patrick September 11, 2020, 5:01pm #2

Whonix doesn’t produce the VirtualBox host software. Neither it does changes to your host.

liam.campbell66:

VD: error VERR_FILE_NOT_FOUND opening image file ‘/Users/liamcampbell/VirtualBox VMs/Whonix-Gateway-XFCE/Whonix-XFCE-15.0.0.9.4-disk001.vdi’ (VERR_FILE_NOT_FOUND).

Means just what it says. That file no longer exists. Possibilities include: you manually deleted these files, another computer user deleted these files, some program such as antivirus deleted that file (although we never had such reports), there are software issues of the host operating system, hardware issues with the physical storage drive or you installed to a device which is no longer connected.

Delete the VM(s) and re-import Whonix.

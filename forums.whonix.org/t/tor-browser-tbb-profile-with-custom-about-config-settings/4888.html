Meanwhile I did some research and answers our question
https://www.askvg.com/list-of-files-that-can-be-hacked-to-configure-firefox/

Following are 3 main files which are used by Firefox:

prefs.js
userChrome.css
userContent.css

All these 3 files are plain text files and are stored in your Firefox profile folder. You can edit them using any standard text editor like Notepad in Windows and gedit in Linux.

Now the question comes what is this Firefox profile folder and where is it stored? Firefox saves all your personalized settings in profile folder and you can find the folder in following location:

Location of Firefox Profile Folder:

~/.mozilla/firefox/xxxxxxxx.default/

More details on making and delelting firefox profiles from the command line

https://support.mozilla.org/en-US/kb/profile-manager-create-and-remove-firefox-profiles

Changes/exceptions are defined in the user.action file which is Privoxy‚Äôs version of config.d

Multiple actions files may be defined in config. These are processed in the order they are defined. Local customizations and locally preferred exceptions to the default policies as defined in match-all.action (which you will most probably want to define sooner or later) are best applied in user.action, where you can preserve them across upgrades. The file isn‚Äôt installed by all installers, but you can easily create it yourself with a text editor.

This is the same problem reported in https://www.whonix.org/forum/index.php/topic,1129.0.html. It has to do with version of the AppArmor profile you are using.

If you want to update Tor Browser with its own updater, you should install the latest profile because TBB 4.5ax creates new updating directories.

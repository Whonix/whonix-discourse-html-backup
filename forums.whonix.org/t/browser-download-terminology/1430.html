Last Signature Creation Date: The cached one. From the file we downloaded last time.
Current Signature Creation: the one of the file that we downloaded just now.

Suggestions on better, less ambiguous wording are welcome. Since that screen is unspecific to Qubes, please post it in the development sub forum.
https://www.whonix.org/forum/index.php/board,5.0.html

All textual strings can be found in this file btw:

   fi
fi

set -o pipefail
set -o errtrace

SCRIPTNAME="$(basename "$BASH_SOURCE")"
IDENTIFIER="torbrowser-downloader"

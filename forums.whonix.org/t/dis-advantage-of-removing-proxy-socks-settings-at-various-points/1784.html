Do you know this page already…?

Whonix

Stream Isolation

Prevent Identity Correlation through Circuit Sharing by using Tor Stream Isolation

Different SocksPorts are pre-configured to implement stream isolation.

Removing proxy settings from Tor Browser might have additional browser fingerprinting issues. (These are documented here- https://www.whonix.org/wiki/Tor_Browser#Local_Connections - and are currently being discussed for improvements - https://phabricator.whonix.org/T455.) Would also break Tor Browser’s tab isolation by socks user name feature ( https://trac.torproject.org/projects/tor/ticket/3455 ).)

Correlation does not imply causation. (http://rationalwiki.org/wiki/Correlation_does_not_imply_causation) I don’t think these events are related. As per Arm FAQ (https://www.whonix.org/wiki/Tor_Controller#Arm_FAQ), it is well known what causes these “duplicates”.

If you have Arm running on the gateway and run the following command 11 times in the workstation.

Then you have 10 [more] duplicates.

Or if you prevent autostart of whonixcheck (https://www.whonix.org/wiki/Advanced_Security_Guide#Prevent_Autostart) (and reboot or stop whonixcheck using “sudo service whonixcheck stop”) then there won’t be any more duplicates.

In short: forget about using Arm for such purposes. It’s really not a tool made for that purpose.

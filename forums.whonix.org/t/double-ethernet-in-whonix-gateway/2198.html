entr0py March 8, 2016, 4:41am #2

If you look at VM Settings for Whonix Gateway in VirtualBox, you’ll see 2 network adapters - one connected to VBox’s NAT router, and one connected to Internal Network ‘Whonix’. eth0 connects to Tor eventually & eth1 connects to Whonix Workstation.

Internet — Host Network Adapter — VBox NAT — eth0 - Whonix GW - eth1 — Whonix WS

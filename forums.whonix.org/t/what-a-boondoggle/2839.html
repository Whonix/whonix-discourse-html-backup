thefoc:

The lastest update broke my Tor install completely.

Please don’t use “Tor” and “Tor Browser” interchangeably. https://www.whonix.org/wiki/Tor_Browser#Tor_vs_Tor_Browser

thefoc:

changes to templates are not being passed on to the child VMs.

thefoc:

In any event, I have to update each new VM manually to 6.0.3.

Correct. This is by Qubes design. Changes to a TemplateVM’s /home/ directory are not passed on to existing child VMs. Tor Browser, as it is installed by Whonix’s tb-updater package is contained entirely within /home/. This means only new VMs will receive the latest Tor-Browser that is installed in the TemplateVM. Existing VMs need to use the internal updater or re-install. https://www.whonix.org/wiki/Tor_Browser#tb-updater_in_Qubes_TemplateVM

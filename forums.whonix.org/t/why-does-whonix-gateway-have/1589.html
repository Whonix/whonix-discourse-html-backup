weareanonymous November 8, 2015, 8:25am #1

Took note of a few interesting features after I imported a sandbox gateway.
Why does Whonix Gateway have the following features in Virtualbox:
-100G HD? Granted, it’s not allocated but 100G for a gateway is still way overkill.
-Floppy and CD/DVD enabled, despite the fact that floppy drives in virtual machines can be serious issues (VENOM for example)?
-Sound card?
-Guest additions? As far as I can tell, they’re not good for much else then sharing clipboard when copying private keys for hidden services. And that doesn’t always work either. Common sense should say that leaving open surfaces to the host system like this can be catastrophic.
-GUI enabled by default?
The gateway should be just that, a gateway which you normally don’t touch unless when you perform updates or run a hidden service.

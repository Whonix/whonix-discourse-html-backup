Patrick April 27, 2016, 5:14pm #2

Debugging advice:
Remove Whonix from the equation. Learn the following first,
sys-net -> sys-firewall -> my Proxy VM -> Tor.

(As per https://www.whonix.org/wiki/Support#Free_Support_Principle.)

Also the following documentation applies to a lesser degree.

  • https://www.whonix.org/wiki/Tunnels/Introduction
  • https://www.whonix.org/wiki/Tunnels/Introduction#Connecting_to_a_tunnel-link_.28proxy.2FVPN.2FSSH.29_before_Tor
  • https://www.whonix.org/wiki/Tunnels/Connecting_to_a_proxy_before_Tor
  • https://www.whonix.org/wiki/Tunnels/Connecting_to_a_proxy_before_Tor#Configure_Whonix-Gateway

The last link being the lest theoretical, most practical. In essence, if latern is listening as a local proxy on 10.137.2.17:8787, you need to configure /etc/tor/torrc on Whonix-Gateway to use it.

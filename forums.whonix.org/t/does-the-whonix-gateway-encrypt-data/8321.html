Support
Aby1 October 21, 2019, 6:57am #1

Does the Whonix gateway encrypt the data that goes through it? So, for example, when Internet traffic data is sent through the gateway, does the data get encrypted?

Patrick October 21, 2019, 8:16am #2

You cannot end-to-end encrypt something at one endpoint if the other endpoint does not support end-to-end encryption.

  • https://www.whonix.org/wiki/Warning#Tor_Exit_Relays_can_Eavesdrop_on_Communications
  • https://www.whonix.org/wiki/Warning#Man-in-the-middle_Attacks
  • https://www.whonix.org/wiki/Warning#Whonix_.E2.84.A2_does_not_Encrypt_Documents_by_Default
  • https://www.whonix.org/wiki/Warning#Whonix_.E2.84.A2_does_not_Clear_Document_Metadata
  • https://www.whonix.org/wiki/Warning#Whonix_.E2.84.A2_does_not_Encrypt_Subject:_and_other_Header_Fields_of_Encrypted_Emails

Itâ€™s only encrypted from Tor running on Whonix-Gateway throughout the Tor network and terminating at the Tor exit relay. What the Tor exit relay can see depends on if end-to-end encryption was used. If you send cleartext, Tor exit can see cleartext. If end-to-end encryption used, Tor exit relay can only see that.

https://www.whonix.org/wiki/Tor_Browser#Encryption contains an image which illustrates that.

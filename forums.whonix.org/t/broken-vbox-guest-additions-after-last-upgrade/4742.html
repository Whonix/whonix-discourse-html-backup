What kind of upgrade was this? To stretch or still only for jessie? The xserver-xorg-core version (>= 2:1.18.99.901) which the guest additions depend on seem to be available only for stretch but not for jessie. There is also no relevant version in jessie-backports. In case some packages are only available from stretch you might want to try this procedure: https://www.whonix.org/wiki/Upgrading_Whonix_13_to_Whonix_14
Note that it is testers only.

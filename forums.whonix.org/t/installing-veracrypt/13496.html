For Qubes-Whonix 16, you want this this package: veracrypt-1.25.9-Debian-11-amd64.deb. Download and perform verification checks as usual in a disposable vm. Qvm-copy the deb installer to your whonix-ws-16 template.

Open a terminal in the same folder you copied the deb installer:
sudo apt update
sudo dpkg -i veracrypt-1.25.9-Debian-11-amd64.deb
sudo apt --fix-broken install

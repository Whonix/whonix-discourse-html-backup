Patrick November 15, 2015, 7:50pm #11

Websites already know you are using Tor, because of your external IP address, which will be Tor exit relays.
Webrtc / local IP addresses are not even needed to find that out.

You’d need a proxy, VPN or ssh to hide Tor from destination websites to begin with:

Whonix

Tunnels/Introduction

Instructions on how to combine tunnels (VPN, SSH, proxy) with Tor. (User -> Tor -> proxy/VPN/SSH -> Internet) (User -> proxy/VPN/SSH ->...

Once you figured that out, that works sufficient for evading general bans of Tor users. But it’s not a bulletproof way to hide Tor. They’d just need to compare ping times, connection throughput and connection flakiness and I don’t think it would be hard to guess with high certainty, that it’s coming from the Tor network.

So what you’re asking for is probably not possible anyhow. And I yet have to hear about websites that go at lengths to use webrtc / local IP analysis to figure out, that one is using Tor.

Just use Tor Browser.

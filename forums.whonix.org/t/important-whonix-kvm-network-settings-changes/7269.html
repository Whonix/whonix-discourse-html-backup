Important: Whonix KVM Network Settings Changes

KVM
important-news
HulaHoop April 30, 2019, 9:11pm #1

Attention new KVM users who’ve downloaded 15.0.0.0.9:

The network import and start instructions have been updated on the wiki for new upcoming versions of Whonix and do not apply to you.

The steps for you are:

virsh -c qemu:///system net-define internal/external_network.xml

virsh -c qemu:///system net-autostart internal/external

virsh -c qemu:///system net-start internal/external

Don’t use both network names in the same command! This is just for brevity.

Versions: >15.0.0.0.9

You will need to delete the old internal and external networks and import the new network settings files which use the new names. If you don’t you will run into conflicts.

To delete the networks go to:
Virtual Machine Manager -> Connection Details -> Virtual Networks

To import the new networks see:

  • https://www.whonix.org/wiki/KVM#Importing_Whonix_.E2.84.A2_VM_Templates

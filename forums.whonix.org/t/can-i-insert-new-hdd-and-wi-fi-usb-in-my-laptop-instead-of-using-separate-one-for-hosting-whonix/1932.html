Patrick January 15, 2016, 6:46pm #4

The reason why new hardware is recommended is that previously used host
operating systems especially such as Windows might have increased the
chances, that the hardware is now infected with a hardware virus. Since
detection and removal of these is next to for any ‘mortal’, your best
bet is starting to use a secure host operating system on more likely
clean hardware.

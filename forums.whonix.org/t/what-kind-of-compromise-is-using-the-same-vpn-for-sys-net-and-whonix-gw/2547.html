As you‚Äôve described:

Your choices for connecting to internet without Tor:

A. user -> isp -> internet

Your choices for connecting to internet via Tor:

  1. user -> isp -> Tor -> internet
  2. user -> isp -> vpn1 -> Tor -> internet
  3. user -> isp -> vpn2 -> Tor -> internet

How to fix Wired Network interface “Device not managed” error in Debian or...

Device not managed error actually is an interesting error. When I tested, Device not managed error only happened to Wired Network interfaces (such as eth0 or eth1) in both Debian and Kali Linux. It…

Enabling Interface Management

If you want Network Manager to handle interfaces that are enabled in /etc/network/interfaces:

Set managed=true in /etc/NetworkManager/NetworkManager.conf. So this file looks like:

[main]
plugins=ifupdown,keyfile

[ifupdown]
managed=true

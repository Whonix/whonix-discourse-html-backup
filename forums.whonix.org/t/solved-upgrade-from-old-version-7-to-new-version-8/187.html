Occq March 21, 2014, 3:28pm #2

Whonix 8 is based on Debian Stable Wheezy (7) and Whonix 7 is/was based on Debian Testing Jessie.

From Whonix 8 release notes:

Sorry, upgrading from existing Whonix version (0.5.6, 7, etc.) to Whonix 8 is not possible.

https://www.whonix.org/blog/whonix-8-released/

You need to download Whonix 8 .ova files to use Whonix 8.

Which tools do you find the need to install in Whonix? Software suggestions and preferences are helpful in planning which software to add to Whonix in the future.

No. Whonix doesn’t modify Tor Browser. That makes a much more robust implementation and is more immune against changes by The Tor Project. Connections to 127.0.0.1:9150 are forwarded to Whonix-Gateway by rinetd. This is described here in detail:

Whonix – 3 May 20

Tor Browser Basics

Using Tor Browser in Whonix ™. Anonymity vs Pseudonymity, HTTPS Everywhere, Torbutton, Protection against dangerous JavaScript, NoScript, Tips, Update Tor Browser, Browser Plugins, Change Language

Also using stream isolation (https://www.whonix.org/wiki/Stream_Isolation).

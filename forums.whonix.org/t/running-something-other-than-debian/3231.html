Running Something Other than Debian?

KVM
HulaHoop December 3, 2016, 3:43pm #1

Whonix KVM is tested with Debian. The install instructions will work flawlessly with other distros for the most part. However note that sometimes some of you who will run into distro-specific quirks like missing dependencies or slightly different Libvirt/KVM package names than those in Debian.

I advise researching how to to install the hypervisor on you system first before opening a support topic. Although I am happy to help, understand that my assistance is limited on that front as I cannot test this on every distro out there.

That window has a timestamp and content once the whonixcheck window is shown is never changed.

Please do what it recommends. Confirm that Whonix-Gateway is running. Then rerun whonxicheck on the workstation. There should be no issue then.

When you restart Whonix-Workstation while Whonix-Gateway is off or not fully booted up, you’ll see that message. That’s expected. To prevent the message, just start Whonix-Gateway. Wait a few moments. Then start Whonix-Workstation. Then again. When you want to restart both, Whonix-Gateway and Whonix-Workstation, it’s expected to see that message if Whonix-Gateway is not fully booted up.

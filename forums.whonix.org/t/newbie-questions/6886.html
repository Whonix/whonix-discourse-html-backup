gentlementle:

Can I change the wall paper of Workstation and/or Gateway to something personal, without compromising anonymity?

No, If the VM was compromised the custom wallpaper could be used as a fingerprint. You should not change settings if the consequences are unknown. You should first ask, like you are doing now

https://whonix.org/wiki/DoNot#Change_Settings_if_the_Consequences_are_Unknown

gentlementle:

If I login to an account, such as email provider, internet forums, reddit, that I‚Äôve logged in before without using TOR, will this then compromise future internet browsing if I wish to maintain being anonymous? If I do so, can I ‚Äúreset a circuit‚Äù to then put me back into being anonymous?

If you logged into those accounts without using Tor you are already de-anonymized. You would never be anonymous while using those identities and/or pseudonyms. So you would have to create new ones. Now, if your Whonix-Workstation was ever compromised while using those de-anonymized personas, any new identities that were created using the compromised VM could be linked to the old ones becuase your activites could be tracked by the malware. This is not saying your external IP address would be leaked though.

https://whonix.org/wiki/DoNot#Login_to_Accounts_Used_without_Tor

Could it be beneficial to use these services where I don‚Äôt have good anonymity, inside a separate Workstation?

It is highly recommended to use a separate Whonix-Workstation for each identity and torified client.

Whonix

Multiple Whonix-Workstations

Compartmentalization. Better separation of different tasks and/or pseudonyms by using multiple Whonix-Workstations.

Patrick January 13, 2016, 12:36am #2

Confirmed. It’s a bug.

To fix run the following commands.

sudo mkdir /var/cache/whonix-setup-wizard/status-files

sudo touch /var/cache/whonix-setup-wizard/status-files/whonix_repository.done

sudo whonixsetup

Will be fixed in Whonix 13.

It does; if the GA are necessary, I try to look for a repository version first before trying the included iso directly from Vbox. Reason being: the package from the distro‚Äôs reposotory seems to just work better. In Debian‚Äôs case, the package they offer (Debian) is a much better fit for a Debian-based virtual machine.

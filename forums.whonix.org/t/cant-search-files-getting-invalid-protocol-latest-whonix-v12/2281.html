Upstream bug report:
https://bugs.debian.org/cgi-bin/bugreport.cgi?bug=766426

Summary, how to fix this:

sudo apt-get install baloo

And then wait a few moments. Search in dolphin will work then.

Will be fixed in Whonix 14 and above.

github.com/Whonix/anon-meta-packages

added baloo to anon-shared-applications-kde 

This fixes dolphin's search function.

Ahmia

Ahmia is a clearnet search engine for Tor's hidden services created by Juha Nurmi. Developed during the 2014 Google Summer of Code with support from the Tor Project, the open source search engine was initially built in Django and PostgreSQL. It collects the peculiar anonymous identifier called .onion URLs from the Tor network and feeds these to its index except those containing a robots.txt file.

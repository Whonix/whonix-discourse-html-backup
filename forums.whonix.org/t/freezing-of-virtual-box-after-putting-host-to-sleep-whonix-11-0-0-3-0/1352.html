Already had some problem with Whonix 10 + putting host into sleepmode (https://www.whonix.org/forum/index.php/topic,1503.0.html). And now having a some problems with Whonix 11.0.0.3.0. When recovering from sleepmode on my host, some times the Gateway (mostly) or the Workstation (once) is completely black and unresponsive.

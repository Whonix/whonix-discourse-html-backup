Whonix for Windows Quick-Start Guide v0.3

  1. Introduction

Whonix for Windows was designed to be a simple introduction, to enable people to experiment with Whonix in a familiar environment without the necessity of complex imports or OS switches.

As you learn how to use Whonix, do not forget that Windows cannot provide a secure foundation for Whonix.

  1. Download, verify and install Whonix

    a) Download the installer. (Whonix-Installer for Windows)
    https://download.whonix.org/windows/Whonix-Installer.exe

    b) Verify the installer.

    Download the signing key [1] and the installer‚Äôs Signature. [2]
    (Verify the Installer).
    https://www.whonix.org/wiki/Installer/Verify_the_Installer_using_the_command_line

    c) Install Whonix.

    Execute the verified Whonix-Installer.exe and follow its instructions. [3]

    Please note: Due to the size of Whonix and the accompanying files, this may take up to half an hour.

    d) Start using Whonix

    You may now enjoy Whonix and all its features, either via Whonix.exe or Whonix for Windows, which you should find on your desktop or VirtualBox.

  2. After installation.

Change passwords for ‚Äúuser‚Äù and ‚Äúroot‚Äù in Whonix-Gateway and Whonix-Workstation. [4]

See Software [5] for a list of pre-installed applications.

See Install Additional Software [6] if necessary.

Regularly check for security updates and apply them. [7]

Always use the Whonix-UI to shut-down Whonix before shutting down the host system.

Read the About [8], Warning [9] and Do Not [10] pages to make sure that Whonix is the right tool for you and you understand well its limitations.

/sys/kernel/debug/ is the usual mount point for it and it‚Äôs already restricted to root in Debian by default so we can‚Äôt do any permission hardening with it in security-misc.

We can‚Äôt just restrict this with apparmor-profile-everything. debugfs doesn‚Äôt have to be at /sys/kernel/debug/. An attacker can mount it anywhere. For example, run

Now there is a debugfs mounted at /mnt. If apparmor-profile-everything denied access to /sys/kernel/debug/, it will do nothing to stop the attacker from just accessing this new mount at /mnt.

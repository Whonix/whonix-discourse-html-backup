I am trying to build Whonix KVM with the recently added serial-console-enable package. I am building on Whonix 15, following these instructions: https://www.whonix.org/wiki/Dev/Build_Documentation/15_full

Command:
sudo ~/Whonix/whonix_build --flavor whonix-gateway-cli --target qcow2 --build

I keep running into an error:
+ error 'Untagged commit! See above!'

Now I am running with --allow-untagged true --allow-uncommitted true flags, and am getting the following error:

last_failed_bash_command: sudo $SUDO_OPTS -E make -f "$make_file" deb-pkg
last_failed_exit_code: 2
ERROR in ././build-steps.d/1200_create-debian-packages detected!

Are there some more detailed build instructions, advice or anything? Thanks.

foieac5 May 26, 2019, 10:41pm #2

Ok I build the gateway. The problem was the RAM on my build machine was too low. I changed it from 2 GB to 8 GB and it worked fine.

Another thing I did, which I think was necessary, was this:

  1. Enable deb-src in /etc/apt/sources.list.d/whonix.list of build machine
  2. sudo apt-get install devscripts debhelper strip-nondeterminism fakeroot apt-transport-tor apt-transport-https python eatmydata aptitude cowdancer

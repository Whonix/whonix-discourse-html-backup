/usr/lib/whonix-libvirt/install is simplistic.

/usr/lib/whonix-libvirt/install started during boot (whonix-libvirt-install.service) after libvirt.

  • during Whonix ISO live boot
  • during first boot of Whonix-Host installed (live [1] or persistent)

And /usr/lib/whonix-libvirt/install is started only once. Because it creates a done file /var/lib/whonix-libvirt/install.done and if that done file exists it won‚Äôt run again. That is to get the initial setup done but from then not interfere with the user.

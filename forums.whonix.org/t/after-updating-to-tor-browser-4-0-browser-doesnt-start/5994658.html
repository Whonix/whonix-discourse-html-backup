Patrick October 21, 2014, 10:29pm #23

The build version is up to date as such, as no issues have been found in these versions that we could not support further by apt updates only without getting a new image. (Whonix 7 -> 8 upgrades as an example was not supported, because too extensive package changes and downgrading from debian testing to stable.)

The whonix-(gateway|workstation)-packages-dependencies 0.4-1 version is also up to date as such as well. It is only a list of packages that are considered essential. When those change and a new one is released, you will be notified. Or when a security upgrade of any of Whonix’s packages will be required, I’ll bump that version number as well.

Unfortunately, we don’t have a mechanism that notifies about arbitrary upgrades Whonix packages. Having a script that extracts latest versions, adds them to Whonix News and makes whonixcheck look at it would be good. If one wanted to help add it, that’s be most appreciated!

pointdrifter:

s there any way to save my bookmarks before I do this

Yes, sure. It’s a Firefox feature inherited by Tor Browser therefore also inherited by Whonix.

Just now documented here:
https://www.whonix.org/wiki/Tor_Browser/Advanced_Users#Backup_and_Restore

Documentation can be certainly followed by advanced users but it’s not novice user proof yet as it does not spell out instructions in a way that doesn’t require any complexity at all.

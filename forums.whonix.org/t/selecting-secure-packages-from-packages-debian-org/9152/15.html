madaidan April 6, 2020, 6:41pm #21
Patrick:

What’s the security issue with dbus more specifically?

D-Bus is an IPC mechanism i.e. apps use it to talk to other apps. It’s possible for an attacker to abuse D-Bus to communicate with an app outside of the sandbox and make it execute commands. For example, https://github.com/netblue30/firejail/issues/796

D-Bus was not created with isolation in mind.

sandbox-app-launcher cuts off access to D-Bus except the per-sandbox session bus to prevent it from being abused for anything.

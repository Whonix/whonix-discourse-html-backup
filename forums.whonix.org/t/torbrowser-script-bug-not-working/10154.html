Patrick August 24, 2020, 1:02pm #3

If there is no user data in Tor Browser that matters… Delete the folder and fix access rights.

sudo rm -r /home/user/.tb
sudo chown --recursive user:user /home/user

Don’t run GUI applications as root as per
https://www.whonix.org/wiki/Root#Graphical_Applications_with_Root_Rights

And “doesn’t work -> try sudo” -> don’t do that. sudo / root is only to be used if that is the sensible thing to do. Otherwise home folder permissions might get broken.

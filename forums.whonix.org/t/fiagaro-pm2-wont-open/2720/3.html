PS1:

The error is unrelated to FPM2. Error is caused by whonixcheck running in the background. See if running FPM in a terminal produces any meaningful output.

FYI anyone who cares, protobuf is Google’s Protocol Buffer parser. It’s most likely installed using Python’s PIP tool.

Set up the windows server however you like. Anonymity of server or not is up to you. Outside of scope for Whonix unless it‚Äôs a Whonix-Custom-Workstation behind a Whonix-Gateway.

To run the client application (VNC client), just run it inside Whonix-Workstation. Proxy settings are nice for stream isolation but that is a minor point compared to possible clearnet leaks when using outside of Whonix.

If the application can connect from inside Whonix-Workstation to the server, then great. It‚Äôs over Tor. If not, then the protocol is using network features such as UDP which are unsupported by Tor. In that case your options boil down to using another application / another protocol or using a workaround to make UDP work over Tor.

Custom user titles are now possible on Whonix.org

News
0brand June 6, 2019, 5:07am #1

Community members might have noticed that a few members of the Whonix team now have custom user titles to the right of their forum user name. This is a cool meta.discourse feature that has been around for a some time, but hasn‚Äôt been offered until now on Whonix.org. Adding a custom user title is a means by which you can receive recognition for your contribution and communicate your ‚Äúrole‚Äù to other in the Whonix community. Hence, the Whonix.org admin is opening up custom user title fields for users that have contributed to Whonix. For the time being, any user that prefers a user title must first have it approved by a forum moderator. This might change in the future at the discretion of the Whonix admin. Community member may use this forum thread to request a custom user title. Before submitting a request, please read the guidelines carefully and remember, with one exception [1] user titles should be thought of solely in terms of your contribution to the Whonix project.

Custom User Title Guidelines

  • You must have contributed to Whonix. For example, answering question on the Whonix forum (forum support), wiki contributions, code contributions, social media contributions, Whonix testing, Reporting Whonix bugs, Squashing Whonix bugs etc.
  • Your contribution has made an impact on Whonix. [2]
  • Not withstanding [1] the title must be related to your Whonix contribution or ‚Äúrole‚Äù. For example, a user contributing code to ‚ÄúHardened Debian‚Äù might use; Whonix ‚Ñ¢ Contributor, Whonix ‚Ñ¢ Kernel Hardening, Whonix ‚Ñ¢ Research‚Ä¶ to name a few possibilities.
  • The custom title must not exaggerate your contribution. For example, requesting a title of Whonix KVM Maintainer after contributing a single line of code.
  • Slang and/or outrageous user titles will not be accepted. For example, titles such as Da Best Wh00nixx Contributorrrrrr will be rejected.
  • We reserve the right to refuse too timely frequent user title change requests.

This thread is designated only for requesting custom user titles. This thread is not for discussing custom user title policy changes. Please keep this thread on topic.

Submitting Your Custom User Title Request

Please take your time when considering a new custom user title. To speed up the process, please include a brief overview of your contribution, unless it is so obvious that it‚Äôs not needed. For example contributors listed on the Team page are well known. A link to said contribution(s) if possible and your preferred custom user title.

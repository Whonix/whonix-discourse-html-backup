Patrick October 7, 2014, 2:22pm #2

Does the Tor Browser Bundle work on the host?

If you have trouble with networking inside Whonix virtual machines, maybe something went wrong while installing VirtualBox. In this case,

Please install some operating system in one of your own virtual machine or download a regular image without Tor enforcement elsewhere. (ex: http://virtualboximages.com/)

Try if networking is functional in the freshly downloaded non-Whonix virtual machine.

If it’s not working: then Whonix won’t work either. You need to sort that out first. Try re-installing VirtualBox. Deinstall VirtualBox, reboot, re-install VirtualBox, reboot, try again.

Patrick October 10, 2014, 3:31am #4
Could a time synchronization issue be causing all of this? My time on the host OS is an hour ahead of the time within Whonix.
Probably not.

Just make sure your date and time on the host is reasonably correct before starting VMs.

Clock inside Whonix VMs are set to UTC by default on purpose, don’t change that. (More info: https://www.whonix.org/wiki/Post_Install_Advice#Network_Time_Syncing) Also no need to change clock in VMs if they are reasonably correct. (Differences because of UTC are okay.)

Pre-Release Audit of qubes-whonix 10.0.4-1

[hr]

Here is a thread for relevant discussion, tickets, links, info, etc for auditing the proposed version 10.0.4-1 of the “qubes-whonix” package for the Qubes + Whonix platform.

The official “qubes-whonix” code repository is here:

  • https://github.com/Whonix/qubes-whonix

This “qubes-whonix” package also uses the QubesBuilder “qubes-builder” and the “qubes-template-whonix” Plugin API:

  • https://www.qubes-os.org/doc/QubesBuilder
  • https://github.com/QubesOS/qubes-builder

Development code repositories are here:

  • https://github.com/nrgaway/qubes-whonix
  • https://github.com/nrgaway/qubes-template-whonix
  • https://github.com/nrgaway/qubes-builder
  • https://github.com/marmarek/qubes-template-whonix
  • https://github.com/marmarek/qubes-builder

Other relevant information to this audit:

  • https://www.whonix.org/blog/pre-release-audit-of-qubes-whonix-10-0-2-1
  • https://phabricator.whonix.org/T275
  • https://www.whonix.org/forum/index.php/topic,1167.0.html

VMware (https://www.whonix.org/wiki/VMware) is unsupported. (https://www.whonix.org/wiki/FAQ#What_do_you_mean_by_unsupported.3F)

I don’t think this problem is caused by rads (https://www.whonix.org/wiki/Desktop#RAM_Adjusted_Desktop_Starter) or too less RAM.

Log into to a virtual console (https://www.whonix.org/wiki/Desktop#Virtual_Consoles). Try manually starting it.

Will probably not work. Maybe this is caused by some incompatibility since Whonix 10 (Debian 7 wheezy based) to Whonix 11 (Debian 8 jessie based).

Check out xorg log. Try also if Debian jessie / KDE works in VMware.

Whonix-I2P - Whonix I2P Integration

We are at a point where some things need to be automated/scripted. I think concentrating on including I2P by default in the Workstation is the easier and higher yield route.

Custom TBB profile for localhost access + Privoxy
https://phabricator.whonix.org/T770

Tor Browser (TBB) profile with custom about:config settings Development
One day in the future if we want to ship with zeronet and/or I2P they need to be readily useful without asking the user to go through and play with about:config settings. The obvious way to get around the fingerprint-ability of allowing localhost access is to enable it only in a custom profile that the user runs at their discretion when needing this functionality. Open question: Is there a mechanism like TBB variables we ca use to readily set these custom options?

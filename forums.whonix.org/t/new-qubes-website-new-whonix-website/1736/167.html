Big update on how the homepage is implemented nowadays…

The homepage that is whonix.org is nowadays also based on MediaWiki. If you are on https://www.whonix.org/ end press edit (using the supermenu).

related:

  • Documentation for Kicksecure ™ Wiki Devs chapter Supermenu and Edit Button in Kicksecure wiki
  • Documentation for Kicksecure ™ Wiki Editors chapter Super Menu in Kicksecure wiki

Much details on homepage:
Documentation for Kicksecure ™ Wiki Devs chapter Homepage (/wiki/Homepage) in Kicksecure wiki

The short summary is, when you edit the homepage whonix.org you’re actually editing whonix.org/wiki/Homepage which mostly uses:

  • Widget:Page Homepage - Whonix
  • MediaWiki:Page Homepage.css - Whonix

There you have the html and CSS.

Since the homepage was specifically prettyfied, it’s mostly HTML / CSS but in the wiki. Due to security reasons and also because it’s difficult (requires HTML and maybe CSS skills), unfortunately it’s not possible to allow edits without prior discussion.

(Whonix is based on Kicksecure.)
(You know that, but other readers might not that’s why I mention it anyhow.)

  4. Deleted a few drivers on my host (keyboard driver was one of them) I didn‚Äôt know about live CDs then so I reinstalled my host and tried to restore my system from backups. Thats when I learned to use a password that i could remember. And not to try to remember multiple complicated passwords.

  5. This next one is pure genius so pay attention!

Nothing to worry about.

Whonix 15 is based on Debian 10 which is now the “oldstable” following the release of Debian 11 “bullseye” on 14 August. The same message appears in standard Debian 10 VM templates.

Debian Project

Bits from the Debian Project

Debian is a free operating system (OS) for your computer. An operating system is the set of basic programs and utilities that make your computer run.

Patrick is busy working on Whonix 16 which is based on Debian 11. Once that is done, we’ll be back on the “stable” release channel.

BTW there is no security risk because Debian has extended support for the old stable releases.

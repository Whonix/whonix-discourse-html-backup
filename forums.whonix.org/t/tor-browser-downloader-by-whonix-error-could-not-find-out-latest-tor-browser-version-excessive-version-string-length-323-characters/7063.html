Tor Browser Downloader by Whonix - ERROR: Could not find out latest Tor Browser version! Excessive version string length. (323 characters)

News
Patrick March 28, 2019, 1:13pm #1

Tor Browser Downloader by Whonix will currently show.

ERROR:
Could not find out latest Tor Browser version! Excessive version string length. (323 characters) Either check.torproject.org has been compromised or this is a Whonix bug.
Possible reasons:

  • The download server is down.

  • File size exceeded (endless data attack triggered).

  • Tor Browser Downloader (by Whonix developers) has been broken due to upstream changes.
    Recommendations:

  • Try again later. If the error persists it probably won’t solve itself before the next update.

  • Check News: https://www.whonix.org/wiki/Stay_Tuned

  • Manually update: https://www.whonix.org/wiki/Manually_Updating_Tor_Browser

This is a known bug. No indicator for a security concern. It has already been reported to Tor Project:

Broken redirect on https://www.torproject.org/projects/torbrowser/RecommendedTBBVersions

Meanwhile please refer to:

Whonix

Manually Downloading Tor Browser

Manually Downloading Tor Browser. Useful in case Tor Browser Downloader (by Whonix developers) is broken.

Patrick March 28, 2019, 1:17pm #2

Advanced users meanwhile could look up the version number here: https://2019.www.torproject.org/projects/torbrowser/RecommendedTBBVersions (or from other sources).

It’s possible to manually set the version you like to download thereby circumventing above bug.

Example stable version at time of writing:

tbb_version=8.0.8 update-torbrowser

Example stable version at time of writing:

tbb_version=8.5a10 update-torbrowser

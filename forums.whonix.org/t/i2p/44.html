Some small manual to simple installing I2P on workstation.

Official instructions here: https://geti2p.net/en/download/debian

First download I2P key wich sighed packeges:

Open synaptic-settings-repositories and add two new distributions:

1)new-binary deb (deb), URL-http://deb.i2p2.no/, distribution-unstable, section- main—(deb http://deb.i2p2.no/ unstable main);

2)new-source-(deb-source), URL-http://deb.i2p2.no/, distribution-unstable, section- main—(deb-src http://deb.i2p2.no/ unstable main).

When you finish with it,reload synaptic that changes applied.After just search i2p,mark and install all packeges.

Start router, i2prouter start.

If someone need it,simplier way to install.

Use nano to edit sources.list.d/i2p.list (from official manual):

nano /etc/apt/sources.list.d/i2p.list

add 2 lines for Whonix stable:

deb http://deb.i2p2.no/ stable main
deb-src http://deb.i2p2.no/ stable main

then press Ctrl+X (ask for saving) press Y,then Enter.Done

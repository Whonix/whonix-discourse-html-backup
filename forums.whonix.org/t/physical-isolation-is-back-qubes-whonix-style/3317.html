For now, the broad steps:

  1. Get 2nd network adapter working in Qubes
  2. No changes to Whonix-GW are necessary.
  3. NetVM-Int: setup static network, forward DNS to GW, add appropriate FORWARD rule
  4. ClientBox: depends

  • working!

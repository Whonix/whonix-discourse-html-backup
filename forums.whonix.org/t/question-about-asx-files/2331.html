There are 3 questions that you are asking:

  1. Was my real IP leaked?
    The reason the answer is ‚Äòno‚Äô is because Whonix Workstation doesn‚Äôt know your real IP address. It only knows what it sees, which is its immediate network surroundings, ie Whonix Gateway.

  2. Was my file‚Äôs metadata leaked?
    First, the obvious reasons that metadata is dangerous is because files can have unique or somewhat unique characteristics, including size, bitrate, codec, gps location, etc. If the filename is ‚ÄúHillary_admits_to_having_sex_with_Monica.mp3‚Äù and that gets out in public, it can be quite revealing in itself.

    VLC‚Äôs default setting is to not send metadata so if you chose not to send metadata in the dialogue, then your file‚Äôs metadata was not sent to begin with.

If you are using User -> Tor -> VPN -> Internet it is expected that you cannot connect to onions.

  • Chapter https://www.whonix.org/wiki/Tunnels/Introduction#Connecting_to_Tor_before_a_tunnel-link_.28proxy.2FVPN.2FSSH.29 mentions that.
  • The comparison table on https://www.whonix.org/wiki/Tunnels/Introduction#Comparison_Table mentions that.

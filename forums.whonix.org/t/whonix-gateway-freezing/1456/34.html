exacto February 3, 2016, 7:48pm #46

There is another issue you should know about I don’t see documented: QXL does not always play nice with kernel modesetting.

On some systems you have to add
qxl.modeset=0
to the kernel command line in
/etc/default/grub
and run
update-grub

It made no difference for this crashing but helps fix some other problems, worth a try.

Patrick January 29, 2016, 9:18pm #4

As the download confirmation screens suggests, choose the lowest advertised version. That usually causes fewest issues.

( https://www.whonix.org/wiki/Tor_Browser#Download_Confirmation_Screen )

Don’t use the hardened version unless you understand that thread. The issue is still active. If it was not, we would update that topic. And if you think it may have been forgotten and want to know if it’s information is still current, please ask in that thread rather than creating a new one.

( Cannot download HARDENED Tor Browser using Tor Browser Updater (Whonix) )

For anything else, or as fallback if still having issues, see the following:

Whonix

Manually Downloading Tor Browser

Manually Downloading Tor Browser. Useful in case Tor Browser Downloader (by Whonix developers) is broken.

( @Ego: Terminology recently has changed. We are calling it now Tor Browser Downloader [by Whonix developers] rather than Updater. There might have been a thread about this, I don’t remember. The reason for this change was basically, that it’s really just a downloader because it cannot keep user profile settings. The updater is the Tor Browser Internal Updater. All the updated terminology and all about the various updating methods can also be found on the https://www.whonix.org/wiki/Tor_Browser page. )

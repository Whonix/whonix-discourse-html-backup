The error message is non-ideal, will be improved in Whonix 14. It just
means Tor Browser exited with code 139. Very much likely not a
tb-starter bug.

Most likely the same would happen, if you started Tor Browser from the
command line. Please try that and see if there are more clues on why it
is crashing.

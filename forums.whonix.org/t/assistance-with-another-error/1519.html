Patrick October 19, 2015, 12:49pm #2

The first one can be ignored. Likely a race condition. The manually started one (?) terminated the autostarted but not yet visible. That message will be removed in Whonix 12.

The second one could be a transient network error? I.e. it will not be reproducible next time? Then ignore it.

The Tor Browser update check in whonixcheck is broken, no longer required because Tor Browser has its own updater. Ignore it. It will be removed in Whonix 12.

anonymous1:

Is there apparmor profile for obfs4proxy?

I don’t think so.

Package obfsproxy owns /etc/apparmor.d/usr.bin.obfsproxy. Related:

  • https://github.com/Whonix/anon-gw-anonymizer-config/blob/master/etc/apparmor.d/local/system_tor.anondist
  • https://github.com/Whonix/anon-gw-anonymizer-config/blob/master/etc/apparmor.d/local/usr.bin.obfsproxy.anondist

Package obfs4proxy owns no file in /etc/apparmor.d/.

  • apt-file list obfsproxy
  • apt-file list obfs4proxy

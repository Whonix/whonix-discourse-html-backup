paradoxum:

On my host machine I’m running the PIA VPN software, so is my traffic going Host > VPN > Whonix/Tor > Web? Is that a good method? Would going through Whonix/Tor then the VPN be better? I am thinking of setting up PiVPN on an RPi and using that as an access point, would that still use the VPN first?

Data flow as follows for VPN on host OS:

User->VPN->Tor->Internet

Another way to look at it ( Whonix-workstation->VPN->Tor->Internet )

  • Keep in mind that your host network traffic is not going through Tor!

  • You asked what is the best VPN setup…What are you trying to accomplish by using a VPN?

https://whonix.org/wiki/Tunnels

https://whonix.orgwiki/Tunnels/Connecting_to_a_VPN_before_Tor

https://whonix.orgwiki/Tunnels/Connecting_to_Tor_before_a_VPN

Also:

Using VPNs with Whonix/Tor can degrade anonymity if used/configured improperly.

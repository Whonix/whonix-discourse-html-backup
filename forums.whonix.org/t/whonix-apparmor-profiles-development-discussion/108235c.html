Normally, Debian should come with AppArmor enabled. They ship some default profiles from their own.

For your profiles, put them in ‚Äò/etc/apparmor.d‚Äô. For each profile, run

Check ‚Äò/etc/default/grub‚Äô. In GRUB_CMDLINE_LINUX_DEFAULT or GRUB_CMDLINE_LINUX, it should read ‚Äúsecurity=apparmor apparmor=1‚Äù. If not. add it, the order does not matter, and run ‚Äòupdate-grub‚Äô.

All the profiles in etc/apparmor.d, if they are not in the disable folder, are loaded at boot.

When using Whonix in Debian, the first package to confine with AppArmor should be VirtualBox: https://www.whonix.org/wiki/AppArmor/VirtualBox. So we (should) have all the network facing packages in whonix apparmored, the virtualization layer with VirtualBox, and the hypervisor itself apparmored.

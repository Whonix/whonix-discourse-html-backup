Yes. This is true for all users - not just bridge users. It means that anyone with access to 10.137.3.1 can route their traffic through those SocksPorts. (A loopback address is something like 127.0.0.1 - available to local machine only.)

If this concerns you (because you connect to Gateway over internet / wifi), you can add authentication / encryption: https://www.whonix.org/wiki/Connections_between_Whonix-Gateway_and_Whonix-Workstation

Someone else can comment on how resistant Qubes is to VM impersonation / VM MITM attacks. My guess would be: relatively safe as long as dom0 is not compromised.

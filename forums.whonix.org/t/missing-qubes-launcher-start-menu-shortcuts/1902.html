This problem maybe fixed by running “Update VM” on whonix-ws and whonix-gw template after installation.

If not, try these commands in dom0 terminal
qvm-start whonix-gw
qvm-sync-appmenus whonix-gw
qvm-start whonix-ws
qvm-sync-appmenus whonix-ws

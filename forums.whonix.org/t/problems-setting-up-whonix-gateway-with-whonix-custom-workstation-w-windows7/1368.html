Did you download the Whonix-Custom-Workstation or manually create it yourself? In other words, did you use:

  • “If you want to manually create a VirtualBox VM (old, stable way), click on expand on the right.” or
  • “If you want to download and import Whonix-Custom-Workstation template (experimental, testers-only), click on expand on the right.”

I speculate the latter. I rather speculate, that the default virtual network card for self created VMs contains a network card supported by Windows while Whonix-Custom-Workstation might have set one that Windows does not support. Please compare those.

Try to change the virtual network card in VM settings. Re-installation of Windows may or may not be required for the new driver to be load.

You not only need to handle the VM folder, but also the settings folder. Some more info on these folders:
https://www.virtualbox.org/manual/ch10.html#idp94907280

I would experiment with setting the environment variable(s). Then move the settings folder to USB. After you figured all of that out, for more convience, you can optionally start VirtualBox using a script located on USB that sets up the environment variable(s) for you.

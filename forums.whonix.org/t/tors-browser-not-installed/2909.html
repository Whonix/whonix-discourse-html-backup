Patrick September 1, 2016, 4:07pm #4
whoarewe:

Does the workstation v13.0.0.1.1 simply not have the Tor Browser installed?

Not installed by default, right.

whoarewe:

Seems odd (to me) that it isn’t.

Yes. Reasons and future:
https://www.whonix.org/wiki/Tor_Browser#Not_installed_by_Default

whoarewe:

Is there a copy of a VBox workstation image with the Tor Browser installed?

No.

There is a bug that slows downloads down. Try this in Konsole:

update-torbrowser --ordinary

Otherwise install manually as per:

Whonix

Manually Downloading Tor Browser

Manually Downloading Tor Browser. Useful in case Tor Browser Downloader (by Whonix developers) is broken.

Restrict Hardware Information to Root - Testers Wanted! News
Please test so perhaps this feature can be enabled by default. Details about your hardware can be used for identification, so Whonix includes the hide-hardware-info.service systemd unit that restricts access to /proc/cpuinfo , /proc/bus , /proc/scsi and /sys to the root user only. This hides most hardware identifiers and increases security as /sys exposes a lot of information that should not be accessible by unprivileged users.

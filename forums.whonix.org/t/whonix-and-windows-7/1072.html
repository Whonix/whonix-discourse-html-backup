More anonymously. Can be more secure depending on threat model, but not necessarily more secure if you don’t know what it does. It’s no magic bullet. See https://www.whonix.org/wiki/Warning especially https://www.whonix.org/wiki/Warning#Man-in-the-middle_attacks.

Do you have Whonix-Gateway running? -> Get it up.

Did you succeed with a usual Whonix setup using a default Whonix-Gateway and a default Whonix-Workstation? -> Get that working first. It will reveal eventual underlying issues.
